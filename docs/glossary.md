# Glossário — Biologia → Técnica

| Termo | Paralelo biológico | Significado técnico |
|---|---|---|
| **Micélio** | Rede subterrânea de fungos | O substrato P2P: conectividade, identidade e economia |
| **Hifa (Hypha)** | Filamento que cresce e se ramifica | Link P2P vivo (QUIC/Noise); fortalece com uso, atrofia sem uso |
| **Anastomose** | Fusão de duas hifas que se encontram | Otimização orgânica de rotas na malha |
| **Corpo de frutificação** | Cogumelo visível (10% do fungo) | Serviço emergente que "brota" do substrato (trait `FruitingBody`) |
| **Sporocarp** | Estrutura que carrega esporos | Volunteer relay (circuit v2) + DNS Spore Bank; crédito ATP; também CDN futuro |
| **Membrana** | Limite célula/ambiente | Papel de rede: `floresta` (IPv6), `raiz` (IPv4+PF), `folha` (outbound), `esporocarp` (relay) |
| **Spore Print** | Esporograma | Mapeamento nome→endereço propagado via gossip (DNS) |
| **Rhizomorph** | Cordão grosso de hifas | Agregação de banda; load balancing por menor resistência |
| **Sclerotium** | Agregado endurecido dormente | Cold storage distribuído que "dorme" e "desperta" |
| **Pheromone** | Sinal químico de reconhecimento | Identidade criptográfica ed25519 + reputação + histórico |
| **Scent** | Cheiro | Score de reputação acumulada (futuro: zk-proof) |
| **Trail** | Trilha de feromônio | Histórico assinado de contribuições |
| **Decay** | Evaporação do feromônio | TTL: identidade esvai se o nó fica inativo |
| **Alarm** | Feromônio de alarme | Sinal de nó malicioso propagado em onda |
| **ATP** | Energia celular | Token de energia (recompensa por CPU) |
| **Enzyme** | Catalisador | Prioridade de acesso (recompensa por RAM) |
| **Mycelia** | Massa de micélio | Direitos de armazenamento futuro (recompensa por storage) |
| **Spore** | Esporo | Reputação e governança (recompensa por banda) |
| **Resilience** | Resistência do organismo | Imunidade a banimento (recompensa por uptime) |
| **Cortex** | Camada externa | Borda da rede: nós leves (IoT, mobile) |
| **Medulla** | Camada interna | Núcleo da rede: nós pesados (servidores, desktops) |
| **Spore Bank** | Banco de esporos | DHT/seed de estado do micélio |
| **Zona de Crescimento** | Região de expansão fúngica | "Região" orgânica que expande/contrai conforme densidade de nós |
| **Biossegundo** | Tempo de sinalização química | Unidade de latência do micélio |
